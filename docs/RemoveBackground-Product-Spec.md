# 在线抠图工具 - Product Spec

## 1. 产品概述

### 1.1 产品定位
一个全品类在线自动抠图工具，支持手动擦除（魔力笔刷）和背景替换（纯色/图片），调用 remove.bg API 实现自动抠图，本地处理手动编辑。

### 1.2 目标用户
- 个人用户：自己使用，避免到处找在线工具

### 1.3 核心价值
- 一键自动抠图：上传图片自动去除背景
- 手动精细调整：魔力笔刷支持擦除和恢复
- 灵活背景替换：支持纯色背景和自定义背景图
- 简洁易用：无需注册，打开即用

## 2. 功能需求

### 2.1 核心功能列表

#### 功能1：图片上传
- **描述**：支持多种方式上传图片
- **输入**：图片文件
- **上传方式**：
  - 拖拽上传：将图片拖拽到上传区域
  - 点击选择：点击上传区域选择文件
  - 粘贴上传：Ctrl+V 粘贴剪贴板图片
- **支持格式**：PNG、JPG、JPEG、WebP
- **文件大小限制**：12MB（与 remove.bg API 限制一致）

#### 功能2：自动抠图
- **描述**：调用 remove.bg API 自动去除背景
- **输入**：上传的图片
- **输出**：透明背景的 PNG 图片
- **AI 能力**：remove.bg API（需用户配置 API Key）
- **处理流程**：
  1. 用户上传图片
  2. 调用 remove.bg API 处理
  3. 返回抠图结果并显示预览
- **免费额度**：每月 50 次（remove.bg 免费额度）

#### 功能3：手动擦除（魔力笔刷）
- **描述**：手动精细调整抠图结果
- **输入**：用户涂抹的区域
- **模式**：
  - 擦除模式：涂抹区域从图片中移除（显示为透明）
  - 恢复模式：涂抹区域恢复到原始状态
- **实现方式**：本地 Canvas 处理（无需调用 API）
- **笔刷大小**：可调节（10px - 100px）
- **交互方式**：鼠标拖动涂抹

#### 功能4：换背景
- **描述**：为抠图后的图片添加背景
- **背景类型**：
  - 纯色背景：选择颜色（支持颜色选择器）
  - 背景图：上传自定义背景图片
- **输出**：合成后的图片
- **应用场景**：
  - 纯色背景：制作证件照、电商主图
  - 背景图：创意设计、产品展示

#### 功能5：图片下载
- **描述**：下载处理后的图片
- **输出格式**：PNG（透明背景）
- **输出方式**：点击下载按钮保存到本地

### 2.2 AI 增强功能

#### AI 增强点：自动抠图
- **原始功能**：图片上传后自动去除背景
- **AI 增强方式**：调用 remove.bg API 进行全品类智能抠图
- **使用场景**：用户上传人像、商品、动物、汽车等图片
- **AI 能力**：remove.bg REST API

## 3. 用户流程

### 3.1 主要用户流程

**流程名称**：在线抠图标准流程

1. **上传图片**
   - 用户操作：拖拽/点击/粘贴方式上传图片
   - 系统响应：显示图片预览
   - 界面状态：上传区域显示图片缩略图

2. **配置 API Key（首次）**
   - 用户操作：点击"开始处理"按钮
   - 系统响应：检测是否有 API Key
   - 界面状态：
     - 有 API Key：直接调用 API 处理
     - 无 API Key：弹出设置弹窗要求输入

3. **自动抠图**
   - 用户操作：点击"开始处理"按钮
   - 系统响应：调用 remove.bg API 处理图片
   - 界面状态：显示处理中的 loading（< 5s）

4. **手动调整（可选）**
   - 用户操作：使用魔力笔刷进行精细调整
   - 系统响应：Canvas 实时处理涂抹区域
   - 界面状态：笔刷模式下显示预览效果

5. **换背景（可选）**
   - 用户操作：选择纯色背景或上传背景图
   - 系统响应：实时合成预览
   - 界面状态：显示带背景的预览效果

6. **下载结果**
   - 用户操作：点击"下载"按钮
   - 系统响应：生成并下载 PNG 文件
   - 界面状态：浏览器下载对话框

### 3.2 异常流程

**异常情况1**：图片格式不支持
- **触发条件**：上传不支持的图片格式
- **处理方式**：显示错误提示，列出支持的格式

**异常情况2**：图片太大
- **触发条件**：上传超过 12MB 的图片
- **处理方式**：显示错误提示，建议压缩后上传

**异常情况3**：API Key 无效
- **触发条件**：remove.bg API 返回认证错误
- **处理方式**：提示 API Key 无效，引导重新设置

**异常情况4**：API 调用次数用尽
- **触发条件**：每月 50 次免费额度用完
- **处理方式**：提示额度用完，引导购买或下月再用

**异常情况5**：抠图失败
- **触发条件**：API 返回处理错误
- **处理方式**：显示"处理失败，请重试"提示

## 5. remove.bg API 能力配置

### 5.1 使用的 AI 能力

| AI 能力 | 用途 | 使用场景 |
|---------|------|----------|
| remove.bg API | 自动抠图 | 用户点击"开始处理"按钮时调用 |

### 5.2 API Key 配置

**配置方式**：用户输入 remove.bg API Key，存储在 localStorage

**交互流程**：
1. 用户首次点击"开始处理"按钮，弹出设置弹窗
2. 用户输入 API Key 并保存
3. API Key 存储到 localStorage，后续请求自动使用
4. 用户可随时通过设置按钮修改 API Key

**API Key 存储**：
- 存储键名：`remove_bg_api_key`
- 存储位置：浏览器 localStorage
- 安全性提示：API Key 仅在本地使用，不上传到服务器

### 5.3 API 调用说明

**调用端点**：`https://api.remove.bg/v1.0/removebg`

**请求方式**：POST

**请求头**：
```
X-API-Key: <用户输入的 API Key>
```

**请求体**：
```
image_file: <图片文件>
size: auto
format: png
```

**响应处理**：
- 成功：返回 PNG 图片数据
- 失败：返回错误信息

### 5.4 免费额度说明

- 免费额度：每月 50 次 API 调用
- 超出后：提示用户额度用完
- 查询余额：可调用 GET /account 端点查询

## 6. 技术需求

### 6.1 技术要求
- **前端框架**：Next.js 15 + React 19 + TypeScript
- **UI 组件库**：Ant Design 5 + Tailwind CSS
- **图片处理**：remove.bg API + Canvas API（本地处理）
- **状态管理**：React Hooks (useState, useRef)
- **部署平台**：Vercel

### 6.2 性能要求
- 图片上传：支持最大 12MB
- API 处理时间：< 5s（根据图片大小）
- 本地处理：< 100ms（笔刷实时响应）

### 6.3 数据要求
- 临时处理：不存储用户上传的图片
- API Key：存储在 localStorage，仅本地使用
- 无需数据库：纯前端工具
